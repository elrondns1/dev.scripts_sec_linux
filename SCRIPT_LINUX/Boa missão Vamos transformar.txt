Boa missão. Vamos transformar isso em algo prático, rápido e reutilizável — no estilo “1 comando e resolveu”.

No Debian 13 você tem **3 níveis de melhoria de imagem**, do simples ao avançado:

---

# 🧭 1. Método rápido (CLI – leve e direto)

Perfeito pra automação e uso diário.

### 📦 Instalar ferramentas:

```bash
sudo apt update
sudo apt install imagemagick
```

### ⚙️ Melhorar qualidade básica:

```bash
magick input.jpg -strip -interlace Plane -quality 90 output.jpg
```

### 🔧 Melhorar nitidez + contraste:

```bash
magick input.jpg -sharpen 0x1.5 -contrast-stretch 0x10% output.jpg
```

👉 Ideal para:

* fotos simples
* reduzir ruído leve
* melhorar contraste

---

# 🧠 2. Método avançado com IA (nível profissional)

Aqui começa a magia real — tipo Gandalf chegando com reforço 😄

## 🔥 Opção 1: Real-ESRGAN (super resolução com IA)

### 📦 Instalar dependências:

```bash
sudo apt install python3 python3-pip git
pip install realesrgan
```

### 🚀 Rodar:

```bash
realesrgan-ncnn-vulkan -i input.jpg -o output.png -s 4
```

📌 O que isso faz:

* aumenta resolução (2x, 4x)
* remove ruído
* melhora detalhes

👉 Ideal para:

* fotos borradas
* imagens antigas
* prints ruins

---

## 🔥 Opção 2: Upscayl (GUI simples e poderosa)

### 📦 Instalar via AppImage:

```bash
wget https://github.com/upscayl/upscayl/releases/latest/download/Upscayl-linux.AppImage
chmod +x Upscayl-linux.AppImage
./Upscayl-linux.AppImage
```

👉 Interface gráfica:

* arrasta a imagem
* escolhe modelo
* clica e pronto

👉 Ideal pra produtividade (regra dos 5 segundos)

---

# 🎨 3. Método visual (edição manual – controle total)

## 📦 Instalar GIMP:

```bash
sudo apt install gimp
```

### 🔧 Ajustes recomendados:

* **Cores → Níveis**
* **Filtros → Melhorar → Reduzir ruído**
* **Filtros → Melhorar → Máscara de nitidez**
* **Filtros → Desfoque → Desfoque gaussiano (leve)**

👉 Ideal para:

* fotos importantes
* ajuste fino manual

---

# ⚔️ Estratégia TARS (melhor resultado possível)

Se quiser qualidade máxima:

```text
1. Upscale (Real-ESRGAN ou Upscayl)
2. Ajustar no GIMP
3. Compressão final com ImageMagick
```

---

# 🧰 Bônus — Script automático (modo guerreiro)

Cria um script pra processar tudo:

```bash
nano melhorar_imagem.sh
```

```bash
#!/bin/bash

INPUT=$1
OUTPUT="melhorada_$INPUT"

magick "$INPUT" -sharpen 0x1.2 -contrast-stretch 0x5% -quality 92 "$OUTPUT"

echo "Imagem gerada: $OUTPUT"
```

```bash
chmod +x melhorar_imagem.sh
./melhorar_imagem.sh foto.jpg
```

---

* Real-ESRGAN → qualidade máxima
* GIMP → refinamento final
