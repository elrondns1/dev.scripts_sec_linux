nano ~/docs/linux/comandos.md

# ## 2. Histórico inteligente do Bash (subestimado, mas poderoso)

# ### Pesquisar comandos já usados
# ```bash
history | grep docker

# Ou interativo:

Ctrl + R

# Melhorar o histórico

# Edite:

nano ~/.bashrc

# Adicione:

HISTSIZE=50000
HISTFILESIZE=100000
HISTCONTROL=ignoredups:erasedups
shopt -s histappend

# Agora seu histórico vira uma base de conhecimento passiva.
