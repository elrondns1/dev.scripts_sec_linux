
#Perfeito 😎 — vamos criar um **script shell simples e seguro** que abre qualquer link **em uma sandbox isolada** no #Linux.

#Ele detecta se você tem o **Firejail** (sandbox leve) ou **Podman/Docker**, e usa o método mais seguro disponível.

#---

## 🧱 **Script: `abrir_link_seguro.sh`**

```bash
#!/usr/bin/env bash
# ==========================================
# abrir_link_seguro.sh
# Abre um link em sandbox segura (Firejail, Podman ou modo texto)
# ==========================================

LINK="$1"

if [[ -z "$LINK" ]]; then
  echo "Uso: $0 <URL>"
  exit 1
fi

echo "[+] Verificando ambiente..."
# Testa se Firejail está instalado
if command -v firejail >/dev/null 2>&1; then
  echo "[+] Usando Firejail com Firefox (sandbox leve)"
  firejail --quiet --private firefox "$LINK"
  exit 0
fi

# Testa se Podman está disponível
if command -v podman >/dev/null 2>&1; then
  echo "[+] Usando navegador em contêiner (Podman)"
  podman run --rm -it --security-opt=no-new-privileges \
    --device /dev/snd --device /dev/dri \
    -v /tmp/.X11-unix:/tmp/.X11-unix:ro \
    -e DISPLAY=$DISPLAY \
    docker.io/jess/firefox "$LINK"
  exit 0
fi

# Testa se Docker está disponível
if command -v docker >/dev/null 2>&1; then
  echo "[+] Usando navegador em contêiner (Docker)"
  docker run --rm -it --security-opt=no-new-privileges \
    --device /dev/snd --device /dev/dri \
    -v /tmp/.X11-unix:/tmp/.X11-unix:ro \
    -e DISPLAY=$DISPLAY \
    jess/firefox "$LINK"
  exit 0
fi

# Caso nada disso esteja disponível, usa modo texto
if command -v w3m >/dev/null 2>&1; then
  echo "[!] Nenhuma sandbox detectada. Usando w3m (modo texto)."
  w3m "$LINK"
elif command -v lynx >/dev/null 2>&1; then
  echo "[!] Nenhuma sandbox detectada. Usando lynx (modo texto)."
  lynx "$LINK"
else
  echo "[x] Nenhum navegador seguro encontrado. Instale Firejail, Podman ou w3m."
  exit 1
fi
```

## 🧩 **Como usar**

1. Salve o script:

   ```bash
   nano ~/abrir_link_seguro.sh
   ```

   (cole o conteúdo acima)

2. Torne executável:

   ```bash
   chmod +x ~/abrir_link_seguro.sh
   ```

3. Rode assim:

   ```bash
   ~/abrir_link_seguro.sh https://example.com
   ```

---

## ⚙️ **Dependências recomendadas**

Instale pelo menos uma das opções abaixo (em ordem de preferência):

### 🔹 Firejail

```bash
sudo apt install firejail
```

### 🔹 Podman (alternativa leve ao Docker)

```bash
sudo apt install podman
```

### 🔹 Navegador em modo texto (backup)

```bash
sudo apt install w3m
```

---

## 🧠 Dica extra

Você pode até criar um **alias** no seu `~/.bashrc`:

```bash
alias abrirseguro='~/abrir_link_seguro.sh'
```

Agora basta rodar:

```bash
abrirseguro https://site.com
```

e o script faz todo o isolamento automático 🔒
